# yummy-time

## Client installation

- Clone the repository:

  ```sh
  git clone git@github.com:yandex-shri-minsk-2016/yummy-time.git
  ```

- Install node and bower dependencies:

  ```sh
  cd client
  npm install
  bower install
  ```
